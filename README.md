# 田舎町・商店前通り v5

外部CDNに依存しない、スマホ縦画面向けのローカル動作版です。

## 反映ポイント
- 開始ボタンの不具合切り分けのため、**外部ライブラリなし**へ変更
- 左右操作の向きを見直し
- 走るを**長押しではなくON/OFF切替**
- コンビニ内装の密度強化
- 町のスポット追加
  - 公衆電話
  - バス停
  - 自販機
  - 掲示板
  - 小橋
  - 神社側分岐
  - 畑
  - 軽自動車
  - 閉店店舗
- 遠景の山と夕方の空気感を強化

## 配置方法
ZIPを解凍し、中身をGitHub Pagesの公開フォルダ直下に配置してください。

## 操作
- 左下スティック: 移動
- 画面右半分ドラッグ: 視点移動
- 右下「走る」: ON/OFF切替
- 右下「調べる」: 近くのスポットを調べる

## 注意
これは**マップと操作感の検証版**です。
テクスチャや照明、店内の小物密度はここからさらに増やす前提です。
