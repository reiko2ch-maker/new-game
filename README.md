# 深夜コンビニ探索 v12

## ファイル構成
- index.html
- styles.css
- main.js

## GitHub Pages反映手順
1. いまの公開ルートにある index.html / style.css / game.js など旧ファイルを削除または退避
2. このZIPを解凍
3. `night_store_v12` フォルダの中身だけをリポジトリ公開ルートに置く
4. GitHub Pages を再読込する
5. キャッシュが残る場合は URL の末尾に `?v=12` を付けて開く

## 操作
- 左下スティック: 移動
- 右半分ドラッグ: 視点移動
- 右下 走る: ON/OFF
- 右下 調べる: 対象に向いて近づいた時だけ有効
- MENU: scanline / 感度 / 歩行揺れ
