# 田舎町・商店前通り v9

スマホ縦画面最優先の、一人称・地方コンビニ探索Webゲームです。

## ファイル構成

- `index.html` : 画面構成
- `style.css` : UI / レイアウト / scanline / HUD
- `game.js` : 描画・操作・マップ・インタラクト・軽量化
- `README.md` : 反映手順と確認ポイント

## GitHub Pages 反映手順

1. 既存の公開先にある `index.html` `style.css` `game.js` を上書きまたは差し替え
2. このフォルダ **中身だけ** を GitHub Pages の公開ルートに置く
3. ブラウザで `https://あなたのURL/?v=9` を開く
4. 古いキャッシュが残る場合は、Safari の再読み込みを実施

## スマホ確認ポイント

- 開いた直後に初期位置から移動できるか
- 左下スティックで前後左右に歩けるか
- 右半分ドラッグで視点が自然に回るか
- 右下 `走る` が ON/OFF トグルで切り替わるか
- コンビニへ入店できるか
- 冷蔵ケースに近づいた時だけ `調べる` が有効になるか
- 飲み物取得後、手元表示が出るか
- レジ周辺を調べられるか
- MENU から scanline の ON/OFF が切り替わるか

## 実装上の方針

- 外部ライブラリなし
- GitHub Pages 直置き対応
- スマホ Safari で重くなりすぎないよう、低解像度レンダリング + アップスケール
- 一人称視点固定
- 店外は暗く、店内は白く明るくして明暗差で怖さを作る
