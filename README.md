- styles.css
- main.js

この3ファイルを GitHub Pages の公開ルートに置いてください。

## 重要
リポジトリ名が `new-game` なら、公開URLは通常  
`https://reiko2ch-maker.github.io/new-game/`  
です。

## 操作
- 左下スティック: 移動
- 右半分ドラッグ: 視点移動
- 右下「走る」: ON/OFF
- 右下「調べる」: 対象に向いて近づいた時だけ有効
- MENU: scanline / 感度 / 歩行揺れ
