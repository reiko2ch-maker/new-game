# 田舎町・商店前通り テストマップ v3

縦画面スマホ向けの一人称ホラー用マップ試作です。  
今回はストーリーを入れず、**歩いて気持ちいい田舎町の空気感**と、**スマホでの操作感**、**店内外の密度感**の確認に絞っています。

## 今回の更新点
- 左右移動が逆に感じやすかった操作を修正
- 走るボタンを**長押し式 → ON/OFF切り替え式**に変更
- コンビニ内装を大幅に密度アップ
- 外観、民家、駐車場、小物を増やして箱っぽさを軽減
- 公衆電話、バス停、閉店店舗、カーブミラー、小さな畑、旅館方面の看板などを追加

## できること
- 自宅前の細い生活道路からスタート
- 商店前通りの小さなコンビニへ移動
- 路地、掲示板、側溝、小橋、自販機、祠側の道などを散策
- 店内の冷蔵庫・レジ・CRT・自販機・掲示板などを「調べる」
- 縦画面スマホ前提の左スティック移動 / 右ドラッグ視点 / 走る切替 / 調べる

## ファイル構成
- `index.html` : UIと画面構成
- `style.css` : 縦画面向けHUDスタイル
- `app.js` : Three.jsでのマップ生成と操作処理

## GitHub Pagesで動かす手順
1. ZIPを解凍
2. 中身をGitHubリポジトリ直下に配置
3. GitHubの `Settings > Pages` を開く
4. Branchを `main` / Folderを `/root` に設定
5. 数十秒待って公開URLにアクセス

## 今回のマップ方針
- 遊べる町ではなく、**歩くだけで夏休み感が出る田舎町**
- 主役はコンビニ周辺
- 重要なのはオブジェクト数より**密度のムラ**
- 光は「コンビニの白」「町灯りの暖色」「路地の暗さ」を分離
- 遠景の山・送電線・家影で箱庭感を減らす
- 近づく場所だけを濃く作り、遠景は軽量に見せる
