# 田舎町・商店前通り v7

スマホ縦画面 / GitHub Pages向けの軽量レイキャスト版です。

## 反映方法
1. 既存の `index.html` `style.css` `game.js` を削除
2. このフォルダの中身をルート直下へ配置
3. GitHub Pages を更新
4. `?v=7` を付けて開く

例:
`https://reiko2ch-maker.github.io/?v=7`

## 操作
- 左下スティック: 移動
- 右半分ドラッグ: 視点移動
- 走る: ON/OFF
- 調べる: 近くのスポット説明
- QUALITY: 低/中/高 切替
