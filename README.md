# 田舎町・商店前通り / 再構築版

縦画面スマホ向けの一人称マップ検証プロトタイプです。

## 特徴
- 外部ライブラリなし
- GitHub Pagesそのまま配置可能
- 左下スティック移動 / 右半分ドラッグ視点移動
- 走るはON/OFF切替
- 調べるボタンでスポット説明
- コンビニ内外・路地・民家・掲示板・公衆電話・バス停・小橋・祠方向・閉店店舗を収録
- 夕方〜夜の空気感を出すための簡易環境音つき（初回タッチ後に再生される場合があります）

## 使い方
1. ZIPを解凍
2. 中身の `index.html` `style.css` `game.js` `README.md` をGitHub Pages公開先のルートへ配置
3. ブラウザで開く

## 反映されないとき
- 古いファイルを削除してから差し替える
- `?v=final` のようにクエリを付けて開く
- ブラウザキャッシュを消す
